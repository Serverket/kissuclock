## :gear: Install & Run  
You'll need at least `node 18`, `electron`, and optionally `bun` (it's faster with it but `npm` also works) to compile everything up:

``` 
bun install  
bun run start  
```

## :star2: Main Features  

**Frontend / UI:**  
* HTML
* CSS
* JavaScript
* Electron
* Cross-platform
* Multi-language support
